# Свадебное приглашение

Готовый одностраничный сайт находится в `index.html`.

## Онлайн-отправка анкеты на GitHub Pages

GitHub Pages публикует только статические файлы, поэтому для приема ответов анкеты нужен внешний сервис: Formspree, Getform или свой API.

1. Создайте endpoint формы в выбранном сервисе.
2. Откройте `script.js`.
3. Впишите ссылку в строку:

```js
const formEndpoint = "https://formspree.io/f/your-form-id";
```

После этого форма будет отправлять ответы онлайн.

## Альтернатива: Netlify Forms

Вариант 1: Netlify Forms

1. Загрузите папку `outputs` на Netlify.
2. Форма уже размечена как `data-netlify="true"`.
3. Ответы появятся в разделе Forms проекта на Netlify.

## Шрифт

В CSS первым указан `English111 Vivace BT`. Если этот шрифт установлен у посетителя, браузер применит его автоматически. Если есть файл шрифта, положите его в `assets` и подключите через `@font-face` в начале `styles.css`.
